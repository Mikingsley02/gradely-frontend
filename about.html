Gradely.info — Smart School Management for Nigerian Schools

Gradely.info is a complete digital platform that helps Nigerian schools automate administration, share results instantly, and communicate seamlessly with parents — all in one secure place.

How it works – Step by step:

  1. Schools sign up and activate features like Financial Records from the admin dashboard using Google Authenticator for added security.
  2. Register students and provide each parent with a unique Student ID + password to access their child’s portal. Parents can change the password later.
  3. Assign students to classes, record attendance, post results, tests, and announcements — all from one dashboard.
  4. Parents get real-time alerts via in-app notifications, SMS, and email whenever assignments, results, or school announcements are posted.

Key benefits:

  • ✅ Instant report cards & automated result sharing
  • ✅ Attendance tracking with automatic parent alerts
  • ✅ Secure financial records with 2FA activation
  • ✅ Bulk student registration & class management
  • ✅ School-wide announcements via SMS/email/in-app
  • ✅ Affordable monthly plans: Basic ₦83,900 | Pro ₦89,700

For schools: Less paperwork, better organization, and stronger parent engagement.

For parents: Always informed — no more waiting for manual reports.
